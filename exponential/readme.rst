Steps for this project to work
1. for this CodeIgniter project to work create a folder in your web server called exponential.
2. change apache port to :60 or change this projects configurations and change them on the view
3. create a database u109936735_badb and import the database in this project.
4. go to your web server and view it, http://localhost:60/exponential/
